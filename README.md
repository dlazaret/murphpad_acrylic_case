# murphpad_acrylic_case
Stacked acrylic case for MechWild MurphPad

## Example configuration
| Layer position | USB cutout | Thickness |
| -------------- | ---------- | --------- |
| Top            | no         | 1 mm      |
| Middle 1       | no         | 3 mm      |
| Middle 2       | no         | 3 mm      |
| Middle 3       | yes        | 3 mm      |
| Middle 4       | yes        | 3 mm      |
| Middle 5       | yes        | 2 mm      |
| Middle 6       | no         | 2 mm      |
| Bottom         | no         | 2 mm      |

### Standoffs
- 5 mm between PCB and bottom layer
- 10-15 mm between top and bottom layer
